# SMART_CLI_CALULATOR
Created a smart CLI calculator just as a mini project

A simple Smart Command Line Calculator built using Python.  
It allows users to perform basic arithmetic operations from the terminal.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User input based CLI interface
- Error handling
